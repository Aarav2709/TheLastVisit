# The Last Visit
- The Last Visit is a short first person narrative experience created for the Silico Battles Inter School Competition, Level Zero.
- You return to your childhood home one last time to collect a set of important documents before the house is cleared. As you explore the house, ordinary objects slowly reveal a story about memory, loss, and the passage of time.

## Theme
This project is inspired by the competition theme **Memento Mori**, exploring the value of memories, the passage of time, and the importance of appreciating life before it becomes a memory itself.

## Development

The game was developed using Roblox Studio, Rojo, Visual Studio Code, and Git for version control.
